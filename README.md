# توثيق AI Camera (عربي)

هذا المجلد يحتوي توثيقًا عمليًا ومفصلًا لمشروعين:

1. [ai-camera-backend-fresh](./ai-camera-backend-fresh.md)
2. [ai-camera-app](./ai-camera-app.md)

## ترتيب القراءة المقترح

1. ابدأ بملف backend لفهم البنية العامة والـ API.
2. انتقل إلى ملف app لفهم تطبيق الموبايل وكيف يستهلك الـ API.
3. استخدم قسم "تكامل backend + app" في كل ملف قبل الاختبار أو التسليم.

## الهدف من هذا التوثيق

- تسريع Onboarding للمطورين.
- توحيد طريقة التشغيل المحلي.
- تقليل أخطاء الربط بين التطبيق والمخدم.
- توضيح المسارات الأساسية بدل الاعتماد على استكشاف الكود فقط.
