المتطلبات الأساسية

  • PHP
  • Composer
  • Node.js و npm
  • قاعدة بيانات (MySQL أو أي محرك يدعمه Laravel)
  • بيئة تشغيل Laravel محلية مثل XAMPP أو Sail أو Valet

إعداد البيئة

افتح ملف .env وحدّث إعدادات بيئتك محليًا. يجب تعديل بيانات قاعدة البيانات يدويًا قبل تشغيل migration. 

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=your_database_name
DB_USERNAME=your_database_user
DB_PASSWORD=your_database_password

إعداد قاعدة البيانات

بعد تعديل .env شغّل:

php artisan migrate --seed

أصول الواجهة الأمامية

npm install
npm run dev

تشغيل الاختبارات

php artisan test --compact

تشغيل خادم التطوير

php artisan serve
