نظرة عامة قصيرة

هذا المشروع هو تطبيق Backend مبني باستخدام Laravel. يوضح هذا الدليل خطوات الإعداد والتشغيل محليًا للمرة الأولى بطريقة واضحة ومنظمة.

حالة الواجهات حسب الدور

سوبر أدمن

إعداد محلي بخطوة واحدة (انسخ وشغّل):

cd ai-camera-backend-fresh
composer install
cp .env.example .env
php artisan key:generate
# عدّل بيانات قاعدة البيانات داخل .env
php artisan migrate --seed
npm install
npm run dev
php artisan test --compact
php artisan serve

أدمن

إعداد محلي بخطوة واحدة (انسخ وشغّل):

cd ai-camera-backend-fresh
composer install
cp .env.example .env
php artisan key:generate
# عدّل بيانات قاعدة البيانات داخل .env
php artisan migrate --seed
npm install
npm run dev
php artisan test --compact
php artisan serve

التطبيق

إليك دليل إعداد كامل خطوة بخطوة للمشروع، بدءًا من استنساخ المستودع وحتى تشغيل التطبيق:

1

استنسخ المستودع:
git clone <your-repo-url>
cd ai-camera-app

2

ثبّت الاعتماديات:
npm install
(أو yarn install إذا كنت تستخدم Yarn)

3

(اختياري) إذا كنت تريد إعادة ضبط المشروع إلى حالة فارغة:
node reset-project.js
(اتبع التعليمات الظاهرة لنقل الملفات الحالية أو حذفها)

4

شغّل خادم Expo للتطوير:
npx expo start

5

افتح التطبيق على جهازك أو المحاكي باستخدام رمز QR أو تطبيق Expo Go.

6

أصبح التطبيق الآن جاهزًا للتطوير والتشغيل. 

git clone <your-repo-url>
cd ai-camera-app
npm install
# أو yarn install
node reset-project.js
npx expo start

إعداد البيئة

افتح ملف .env وحدّث إعدادات بيئتك محليًا. يجب تعديل بيانات قاعدة البيانات يدويًا قبل تشغيل migration. 

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=your_database_name
DB_USERNAME=your_database_user
DB_PASSWORD=your_database_password

إعداد قاعدة البيانات

بعد تعديل .env شغّل:

php artisan migrate --seed

أصول الواجهة الأمامية

npm install
npm run dev

تشغيل الاختبارات

php artisan test --compact

تشغيل خادم التطوير

php artisan serve

الحالة الحالية / القيود

  • صفحات الأدمن: تمت إضافة إعداد بخطوة واحدة
  • صفحات السوبر أدمن: تمت إضافة إعداد بخطوة واحدة
  • صفحات التطبيق: تمت إضافة دليل إعداد كامل

أصبحت أقسام Super Admin و Admin تحتوي على أوامر الإعداد الكاملة، كما تم إضافة دليل إعداد كامل لقسم التطبيق.

استكشاف الأخطاء وإصلاحها

مشكلة في مفتاح التطبيق

php artisan key:generate

مشكلة في قاعدة البيانات

  • تأكد من صحة القيم داخل .env.
  • تأكد أن خدمة قاعدة البيانات تعمل.

مشاكل في أصول الواجهة

npm install
npm run dev

فشل في Migration

  • تأكد من وجود قاعدة البيانات.
  • تحقق من صلاحيات المستخدم.
